Gibson 계열 기타를 중고로 볼 때 헷갈리기 쉬운 **바디 종류, 대표 라인, 넥감, 픽업 성향**을 빠르게 비교하는 페이지입니다.
모델명만 보고 판단하기보다 연식, 실제 장착 픽업, 넥 프로파일, 전장 상태, 수리 이력을 함께 확인하는 것을 전제로 합니다.

## 핵심 요약

- **Les Paul:** 두껍고 묵직한 중저역, 긴 서스테인, 압축감 있는 험버커 톤이 기준입니다.
- **SG:** 가볍고 상위 프렛 접근성이 좋으며, Les Paul보다 빠르고 공격적으로 반응하는 편입니다.
- **ES / Semi-hollow:** 공명감과 따뜻함이 커서 클린, 크런치, 블루스, 재즈 쪽에 강합니다.
- **Junior / Special:** 장식보다 단순한 구조와 직접적인 반응을 중시합니다. P-90 모델이 많습니다.
- **넥감:** 50s Vintage, Rounded, SlimTaper, Modern 계열처럼 손에 차는 정도와 어깨감이 다릅니다.
- **픽업:** P-90은 거칠고 선명한 싱글코일 계열, PAF 계열 험버커는 Gibson 빈티지 톤의 중심입니다.

## Gibson 넥감과 프로파일 {#gibson-neck-profiles}

  caption="Gibson 넥감은 두께뿐 아니라 어깨감, 테이퍼, 힐, 지판 반경이 함께 만듭니다."
%}

Gibson 넥감은 단순히 “두껍다” 또는 “얇다”로만 끝나지 않습니다. 손바닥에 걸리는 **어깨감**, 1프렛에서 12프렛으로 갈수록 두꺼워지는 **테이퍼**, 상위 프렛 쪽의 **힐 구조**, 지판의 **반경과 프렛 크기**, 마감의 끈적임까지 합쳐져 실제 체감이 달라집니다. 그래서 같은 Les Paul이라도 Standard 50s와 Standard 60s는 손에 잡히는 느낌이 꽤 다릅니다.

대부분의 현대 Gibson USA 모델은 24.75인치 스케일을 기본으로 보지만, 넥 프로파일은 모델별로 다릅니다. Les Paul Standard 50s는 공식 사양에서 50s Vintage 프로파일을 쓰는 대표 예이고, Les Paul Standard 60s와 ES-335 60s는 SlimTaper 계열로 더 얇고 빠른 편입니다. SG Standard는 Rounded 프로파일이라 Les Paul 50s처럼 아주 두툼한 빈티지 감각보다는 둥글고 안정적인 C형에 가깝게 느껴집니다. Les Paul Modern 계열은 SlimTaper 성향에 Modern Contoured Heel, 복합 지판 반경 같은 현대식 연주성을 더한 쪽입니다.

<div class="table-scroll plain-table" markdown="1">

| 넥 표기 | 손에 잡히는 느낌 | 대표적으로 볼 모델 | 잘 맞는 경우 | 중고 확인 포인트 |
|---|---|---|---|---|
| 50s Vintage / Rounded C | 손에 꽉 차는 두꺼운 C형, 어깨감이 있음 | Les Paul Standard 50s, 일부 50s 리이슈 계열 | 코드, 리듬, 빈티지 Les Paul 감각을 선호 | 1프렛과 12프렛 두께, 손이 피로한지 확인 |
| Rounded | 둥글고 안정적인 C형, 과하게 얇지 않음 | SG Standard, 일부 Studio/Tribute 계열 | 얇은 넥은 불안하지만 50s Les Paul은 부담스러운 경우 | 넥다이브와 함께 스트랩 착용감 확인 |
| SlimTaper / 60s SlimTaper | 낮고 빠른 느낌, 포지션 이동이 쉬움 | Les Paul Standard 60s, SG '61, ES-335 60s | 리드, 벤딩, 빠른 포지션 이동을 자주 하는 경우 | 너무 얇게 느껴져 손에 힘이 들어가는지 확인 |
| Modern SlimTaper / Modern Contoured Heel | 얇은 계열에 상위 프렛 접근성 강화 | Les Paul Modern, Les Paul Modern Figured | 현대적인 범용성, 코일스플릿, 상위 프렛 사용 | 힐 컷, 복합 반경, 전장 기능 정상 작동 확인 |

</div>

넥감을 빠르게 고르면, 두꺼운 넥과 묵직한 코드감을 좋아하면 **Standard 50s**, 얇고 빠른 연주감을 원하면 **Standard 60s**, 가볍고 둥근 SG 감각을 원하면 **SG Standard**, 상위 프렛 접근성과 현대 기능까지 원하면 **Les Paul Modern** 쪽을 먼저 잡아보는 흐름이 좋습니다.

중고 매물에서는 가능하면 판매자에게 1프렛과 12프렛 두께를 물어보되, 숫자만으로 결론내리지는 마세요. 비슷한 두께라도 어깨가 넓으면 더 두껍게 느껴지고, 마감 상태나 프렛 크기에 따라 손맛이 달라집니다. 특히 헤드/넥 수리 이력, 트러스로드 여유, 프렛 마모는 가격과 연주성에 바로 영향을 줍니다.

픽업은 커버만 보고 판단하기 어렵습니다. 중고에서는 실제 장착 픽업, 배선 상태, 납땜 흔적, 픽업 뒷면 라벨을 함께 확인해야 합니다. 같은 험버커 크기라도 '57 Classic, BurstBucker, 490/498, 496/500T는 출력과 어택이 다르게 느껴집니다.

|---|---|---|---|
| P-90 | 굵고 중역이 두꺼운 싱글코일 계열 | Junior, Special, 펑크, 블루스록, 개러지 | 험버커보다 노이즈가 있음 |
| 490R / 490T | Gibson식 범용 험버커 | SG Standard, 범용 록 사운드 | 빈티지 약출력도, 메탈 고출력도 아닌 중간 성향 |

| 496R / 500T | 세라믹 고출력 험버커 | 하드록, 메탈, 강한 어택 | 빈티지 블루스 PAF 톤을 원하면 과할 수 있음 |
| Firebird 픽업 | 밝은 고역과 타이트한 저역 | Firebird, 슬라이드, 블루스록 | 일반 미니 험버커와 구조/성향을 구분 |

## 빠른 선택 가이드

- **따뜻한 빈티지 Les Paul 톤:** '57 Classic, BurstBucker 1/2, Custombucker.
- **선명한 현대 PAF 톤:** BurstBucker Pro, BurstBucker 61.
- **범용 Gibson 록 사운드:** 490R/490T, 490R/498T.
- **거칠고 직접적인 록 사운드:** P-90.
- **하이게인/메탈 성향:** 496R/500T.
- **세미할로우의 명료함:** T-Type.
- **밝고 독특한 Firebird 사운드:** Firebird 픽업.

## 중고 구매 체크 포인트

1. 모델명보다 **연식별 공식 스펙**을 먼저 확인합니다.
2. 픽업 커버만 보고 판단하지 말고, 실제 장착 픽업과 배선 상태를 확인합니다.
3. Gibson 계열은 헤드/넥 수리 이력이 가격에 크게 반영됩니다.
4. SG와 Firebird는 스트랩을 메고 밸런스와 넥다이브를 확인하는 편이 좋습니다.
5. 넥 프로파일은 50s/60s 표기만 보지 말고 실제 손에 잡히는 두께와 어깨감을 확인합니다.
6. P-90 모델은 노이즈가 정상 범위인지, 접지/전장 문제가 아닌지 구분해야 합니다.
7. Les Paul은 실중량과 웨이트 릴리프 여부가 연주 피로도에 직접 영향을 줍니다.
