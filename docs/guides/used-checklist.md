# 중고 레스폴 타입 기타 체크리스트

## 1. 거래 전 확인할 질문

판매자에게 아래 내용을 먼저 확인합니다.

```text
넥 휨, 프렛버징, 헤드/넥 크랙, 전기계통 문제는 없을까요?
넥/브릿지 픽업과 셀렉터, 볼륨/톤이 모두 정상 작동하는지도 궁금합니다.
실중량을 알 수 있으면 확인 부탁드립니다.
```

## 2. 구조 확인

- 헤드 앞면과 뒷면에 금이 없는지
- 너트 주변에 금이 없는지
- 헤드와 넥이 만나는 부분에 수리 흔적이 없는지
- 넥 접합부에 크랙이 없는지
- 바디 뒷면과 측면에 큰 충격 흔적이 없는지

## 3. 넥과 프렛

- 줄높이가 과하게 높지 않은지
- 1~5프렛 코드에서 버징이 심하지 않은지
- 7~12프렛 단음에서 음이 죽지 않는지
- 하이프렛 벤딩 시 음이 죽지 않는지
- 프렛 홈이 깊게 파이지 않았는지
- 프렛 끝이 손에 걸리지 않는지

## 4. 전기계통

가능하면 앰프 또는 휴대용 앰프/헤드폰 앰프로 확인합니다.

- 넥 픽업 출력 정상
- 브릿지 픽업 출력 정상
- 3단 셀렉터 정상
- 볼륨 노브 정상
- 톤 노브 정상
- 잭 접촉 불량 없음
- 심한 지직거림 없음

## 5. 하드웨어

- 튜너가 헛돌지 않는지
- 브릿지 포스트가 휘지 않았는지
- 테일피스 포스트가 흔들리지 않는지
- 너트 홈이 깨지지 않았는지
- 스트랩핀이 헐겁지 않은지

## 6. 감점 요소

| 감점 요소 | 판단 |
|---|---|
| 단순 생활기스 | 큰 문제 아님 |
| 바디 작은 덴트 | 가격에 반영하면 가능 |
| 하드웨어 금장 변색 | 큰 문제 아님 |
| 케이스 없음 | 2~5만 원 감점 |
| 전기계통 잡음 | 수리비 감안 |
| 픽업 한쪽 안 나옴 | 원인 확인 전까지 큰 감점 |
| 넥 수리 이력 | 저가 연습용이라도 신중 |
| 헤드 크랙 | 원칙적으로 패스 |
| 트러스로드 문제 | 패스 |
