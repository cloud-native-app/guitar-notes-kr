# 중고가와 재판매 기준

## 중고가를 볼 때의 기본 원칙

저가 레스폴 타입 기타는 5만 원 차이가 큽니다.  
구매할 때는 “나중에 얼마에 팔 수 있나”보다 “내가 편하게 쓸 수 있나”가 먼저입니다.

## 가격대별 손실 감각

| 구매가 | 빠른 재판매 예상 | 손실 감각 |
|---:|---:|---|
| 10~15만 원 | 8~12만 원 | 작음 |
| 16~25만 원 | 12~20만 원 | 감수 가능 |
| 26~35만 원 | 20~30만 원 | 상태 유지 중요 |
| 36~50만 원 | 30~40만 원 | 구매층 좁아짐 |
| 50만 원 이상 | 모델별 편차 큼 | 방치형 목적과 멀어짐 |

## 좋은 중고 거래의 형태

가장 좋은 거래는 다음과 같습니다.

- 신품가 40만~60만 원대 모델을 중고 20만~30만 원대에 구매
- 수리 필요 없음
- 넥/프렛/전기계통 정상
- 거래 지역이 가까움
- 케이스 포함
- 나중에 5만~10만 원 손실로 정리 가능
