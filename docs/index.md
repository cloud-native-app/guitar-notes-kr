## 이 사이트의 정리 기준

<div class="source-box">
  <p>Guitar Notes KR은 특정 매물을 추천하거나 판매자를 평가하지 않습니다. 공식 제품 페이지, 유통사 정보, 공개 가격, 모델 스펙, 실제 구매 체크포인트를 구분해 정리합니다.</p>
  <ul>
    <li>공식 스펙은 브랜드와 제조사 페이지를 우선합니다.</li>
    <li>가격은 작성 또는 수정일 기준 참고값으로 봅니다.</li>
    <li>중고가는 상태, 연식, 색상, 구성품, 지역, 환율에 따라 달라질 수 있습니다.</li>
    <li>판매자 사진, 연락처, 매물 원문 캡처는 저장하지 않습니다.</li>
  </ul>
</div>
