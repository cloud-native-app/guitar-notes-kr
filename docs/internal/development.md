# 개발 메모

이 문서는 사이트 독자용 화면이 아니라 저장소 운영자를 위한 메모입니다.

## GitHub Pages 설정

1. 저장소의 `Settings`로 이동합니다.
2. `Pages` 메뉴에서 Source를 `Deploy from a branch`로 둡니다.
3. Branch는 `main`, folder는 `/docs`로 지정합니다.

## 라이선스 메모

공개 자료로 계속 운영하려면 `CC BY-NC 4.0` 또는 `MIT License` 중 하나를 선택해 `LICENSE` 파일을 추가하는 것을 검토합니다.
