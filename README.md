# Guitar Notes KR

일렉기타를 고르고, 비교하고, 오래 쓰기 위한 한국어 기타 노트입니다.

레스폴 타입 기타, Gibson·Epiphone 모델 차이, 픽업 성향, 중고 구매 체크리스트를 사진과 표, 출처와 함께 정리합니다. 특정 매물을 평가하기보다, 기타를 사기 전에 무엇을 비교해야 하는지 차분하게 볼 수 있는 기준을 모읍니다.

## 다루는 주제

- 레스폴 타입 일렉기타 모델 비교
- Gibson·Epiphone 라인업과 세대별 차이
- P-90, PAF 계열 험버커, BurstBucker 등 픽업 성향
- 중고 거래 전 확인할 넥, 프렛, 전기계통, 수리 이력
- 가격대별 입문용·세컨 기타 후보

## 정리 기준

가격은 작성 또는 수정일 기준의 참고값입니다. 실제 신품가와 중고가는 환율, 수입 시기, 색상, 재고, 할인, 구성품, 상태에 따라 달라질 수 있습니다.

모델 스펙은 연식, 생산국, 리비전, 유통사, 한정판 여부에 따라 달라질 수 있으므로 실제 구매 전에는 공식 제품 페이지와 판매자 실물 사진을 함께 확인하는 것을 전제로 합니다.

특정 판매자, 매물 원문, 연락처, 판매자 사진은 저장하지 않습니다. 특정 매물을 비난하지 않고 일반적인 구매 판단 기준과 체크리스트를 정리합니다.
