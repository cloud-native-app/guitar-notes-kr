# Guitar Notes KR

일렉기타를 고르고, 만지고, 오래 쓰기 위한 한국어 기타 노트입니다.

이 저장소는 레스폴 타입 기타, 픽업, 중고 거래, 모델별 스펙 차이, 구매 체크리스트를 블로그 글처럼 정리합니다. 특정 매물을 평가하기보다, 기타를 사기 전에 무엇을 비교해야 하는지 차분하게 볼 수 있는 기준을 모아 둡니다.

## 이런 기준으로 씁니다

가격은 작성 시점의 참고값으로만 다룹니다. 실제 신품가와 중고가는 환율, 수입 시기, 색상, 재고, 할인, 구성품, 상태에 따라 계속 달라집니다.

모델 스펙은 연식, 생산국, 리비전, 유통사, 한정판 여부에 따라 달라질 수 있습니다. 글에서는 큰 판단 기준을 정리하고, 실제 구매 전에는 공식 제품 페이지와 판매자 실물 사진을 함께 확인하는 것을 전제로 합니다.

특정 판매자, 매물 원문, 연락처, 판매자 사진은 저장하지 않습니다. 특정 매물을 비난하지 않고, 일반적인 구매 판단 기준과 체크리스트를 남깁니다.

## GitHub Pages 설정 메모

1. 저장소의 `Settings`로 이동합니다.
2. `Pages` 메뉴에서 Source를 `Deploy from a branch`로 둡니다.
3. Branch는 `main`, folder는 `/docs`로 지정합니다.
