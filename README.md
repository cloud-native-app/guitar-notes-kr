## 문서 운영 원칙

- 가격은 작성 시점의 참고값이며, 실제 신품가와 중고가는 계속 변동됩니다.
- 특정 판매자, 매물 원문, 연락처, 판매자 사진은 저장하지 않습니다.
- 특정 매물을 비난하지 않고, 일반적인 판단 기준과 체크리스트를 정리합니다.
- 모델 스펙은 연식, 생산국, 리비전, 유통사에 따라 달라질 수 있습니다.

## GitHub Pages 설정

1. 저장소의 `Settings`로 이동합니다.
2. `Pages` 메뉴에서 Source를 `Deploy from a branch`로 둡니다.
3. Branch는 `main`, folder는 `/docs`로 지정합니다.
